Trying Out Red Mine

Jan 9th, 2013

I was trying out this [tutorial] (http://carolhsu.github.com/blog/2012/11/05/how-to-deploy-redmine-at-heroku/), which worked well. The only change I made was using mysql instead of mysql2 in database.yml setting, and using 2.2-stable branch.

I also placed a few themes in public/themes, which you can switch in Administrator/settings/display.

Deploying with Heroku the first time was interesting. All you need to do is installing the Heroku toolbelt, then, you can use heroku login to switch to heroku env., and heroku keys:add ~/.ssh/id_rsa_x.pub to setup key for ssh session when doing git push.

Note that since Heroku does not use sqlite3 anymore, so sqlite3 related gems need to be removed in Gemfile.
